THEOlive offers a number of locations where you can create your channels

---

THEOlive allows channels to be created in different regions (also referred to as "ingest locations"). Selecting an ingest location close to you will bring the lowest possible latency to your viewers all over the world.

You can find an overview of the possible regions in the table below.

| Code          | Location       |
| :------------ | :------------- |
| `europe-west` | Ireland        |
| `us-west`     | Oregon         |
| `us-east`     | North Virginia |
